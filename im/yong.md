#小小输入法

## 安装 (Ubuntu 16.04 LTS)

```shell
# cd /usr/lib/x86_64-linux-gnu/qt5/plugins/platforminputcontexts/
# cp ~/libyongplatforminputcontextplugin.so ./
# chmod +x libyongplatforminputcontextplugin.so
# cd /opt/
# cp -r yong ./
# reboot
```

`Ctrl + Space` 激活输入法

## 修改皮肤

```shell
$ cd ~/.yong/
$ mkdir skin
$ cp -r ~/Dark skin
```
